# [bash 使用]()

## 创建批处理文件 **test.sh**

```shell
#! /bin/bash

echo "hello world!"
```

## 执行批处理文件 **bash**

方式一：

```shell
bash test.sh
```

方式二：

```shell
chmod +x test.sh    # 使 test.sh 具有可执行的权限

./test.sh           # 执行 test.sh 脚本
```
